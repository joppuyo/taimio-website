Open source

Taimio is free and open source software released under MIT license. Follow development on GitHub.

Hosting

Everyone is welcome to use Taimio here at app.taim.io for free but private self-hosting is always possible.

REST API

Taimio provides a comprehensive REST API for creating new client applications and scripts for reporting to other systems. Your data will never be locked in Taimio.
